Aegis Linux is a modern, open-source system optimization, monitoring, maintenance and security suite designed specifically for Linux.

Inspired by the best ideas from tools like CCleaner, BleachBit, Glances, Stacer and Microsoft Sysinternals, Aegis Linux combines performance optimization, real-time monitoring, diagnostics and security auditing into a single application.
